CONTRIBUTING
==

Before commit, push or pull request.
--
```sh
make test && php README.md
```

Start developing.
--
We need:

1. Git :) and clone this repo.
2. PHP >=5.3
3. make (`apt-get install make`)
4. [composer](https://getcomposer.org/) and `composer install`.

You may have:

1. [EditorConfig](http://editorconfig.org/)
